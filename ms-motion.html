MSMotion

Advanced layered motion composition & animation retargeting system for Unity.

Master Your Character Animations

MSMotion is a cutting-edge animation and motion composition system designed for Unity. It enables developers to create complex, high-fidelity humanoid motion systems using layered overrides, procedural IK, and trajectory-based motion mapping. It integrates natively with Unity's Animation Rigging for seamless transitions and professional-grade character control.

Intuitive & Powerful Features

Layered Motion Composition

Compose complex animation sequences using layered overrides. Blend and stack animations with precise weight controls.

Built-in & Procedural IK

Dynamic Foot IK and Full-Body IK solvers designed to prevent foot sliding, adapt to uneven terrain, and maintain posture.

Animation Rigging Integration

Natively integrates with Unity's Animation Rigging package to dynamically fade rig weights, ensuring perfectly smooth transitions.

Trajectory-based Root Motion

Control and redirect character root motion dynamically along custom trajectories, giving you unparalleled locomotion control.
