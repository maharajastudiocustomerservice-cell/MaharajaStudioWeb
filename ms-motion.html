MSMotion

Connect animations, visual effects, physics, and gameplay logic on a single shared timeline in Unity.

Animate. Simulate. React. All in One Place.

MSMotion is a comprehensive motion and event composition system that brings your character animations, visual effects, secondary physics, and gameplay logic together onto a single shared timeline. With real-time scene preview and zero performance cost at runtime, MSMotion enables you to author and preview frame-accurate character actions entirely within the Unity Editor without entering Play Mode.

Intuitive & Powerful Features

Layered Motion & Modifier Nodes

Compose complex sequences using layered overrides and additive blending. Add spring-mass Jiggle Physics for capes, hair, and chains, plus Look-At tracking and spline trajectories.

FX & Gameplay Event Lanes

Trigger pooled audio, particle systems, VFX Graph parameters, and Cinemachine Camera Shakes frame-accurately on the timeline, scaling intensity dynamically during crossfades.

Built-in Clip Editor

Extract FBX animations, crop/scale ranges, smooth curves, remove redundant keyframes, and mirror clips non-destructively in the editor with ghost preview support.

High-Performance Runtime

Powered by Unity's low-level PlayableGraph API with zero GC spikes. Features pooled audio/particle effects and robust, state-query APIs for precise gameplay integration.
