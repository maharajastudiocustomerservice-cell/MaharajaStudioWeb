Object Finder Pro

The ultimate Unity editor tool to find any object or asset in your project instantly.

Stop Searching, Start Creating

Object Finder Pro transforms your workflow by turning your entire Unity project into a searchable database. No more wasting time digging through complex hierarchies or endless project folders. With a powerful query language and asynchronous searching, you get the exact results you need, right when you need them.

Get on Asset Store Read Docs

Power-Packed Features

Smart Search Query Language

Go beyond simple name searches. Filter by component properties (e.g., `comp(Light).intensity:>5`), tags, layers, memory size, and more.

Asynchronous Engine

Never freeze your editor again. Searches run in the background, keeping your workflow fluid and responsive, even in massive projects.

Find All References

Instantly find every usage of a specific component instance or asset across scenes, prefabs, and ScriptableObjects.

Multi-Component & Signature Filters

Locate GameObjects that have a specific combination of components, or match an exact component signature for template verification.
